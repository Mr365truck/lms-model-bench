Implement a Python class LRUCache that supports the following operations:
1. LRUCache(capacity: int): Initialize the LRU cache with positive size capacity.
2. get(key: int) -> int: Return the value of the key if the key exists, otherwise return -1.
3. put(key: int, value: int) -> None: Update the value of the key if the key exists. Otherwise, add the key-value pair to the cache. If the number of keys exceeds the capacity from this operation, evict the least recently used key.

Both get and put operations should run in O(1) time complexity on average.

Please return only the python code inside a ```python ... ``` block, starting with the class definition. No explanations.
