A trie (pronounced as "try") or prefix tree is a tree data structure used to efficiently store and retrieve keys in a dataset of strings. There are various applications of this data structure, such as autocomplete and spellchecker.

Implement the Trie class:
1. Trie(): Initializes the trie object.
2. insert(word: str) -> None: Inserts the string word into the trie.
3. search(word: str) -> bool: Returns true if the string word is in the trie (i.e., was inserted before), and false otherwise.
4. startsWith(prefix: str) -> bool: Returns true if there is a previously inserted string word that has the prefix prefix, and false otherwise.

Please return only the python code inside a ```python ... ``` block, starting with the class definition. No explanations.
